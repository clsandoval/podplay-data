# Pickleball Consulting

Pickleball facility consulting for the Philippines market. Modeled on JDC Pickleball's $6,000 USD/month retainer that Kosmas itself pays — the inversion is "we've been through it, we're on the ground, same timezone, local network, so we can do this for PH operators ourselves."

**Status:** Stub — menu sketched (`cs/monorepo/projects/kosmas/kosmas-pickleball-consulting-menu.md`), engagements not yet tracked here.

## Engagement tiers (from menu)

- **Tier 1 — Feasibility & Pre-Build** — one-time, ₱150K–250K. Market analysis, site evaluation, financial model, sizing, go/no-go deck.
- **Tier 2 — Design & Planning** — one-time, ₱200K–350K. Layout, surface/lighting spec, vendor intros, permitting.
- **Tier 3 — Ongoing Advisory** — monthly retainer, ₱60K–100K. Performance review, rate optimization, programming, vendor management, on-call.
- **À la carte** — tournament design, coach certification, mystery shopper.
